## Python `UTF-8` encode

> Python 3.x

```py
#!/usr/bin/python3

# coding=utf-8


```


```py
#!/usr/bin/python3

# coding: utf-8

```

> Python 2.x

```py
#!/usr/bin/python2

# -*- coding: utf-8 -*-


```


```py
#!/usr/bin/python2

# -*- coding: UTF-8 -*-


```
